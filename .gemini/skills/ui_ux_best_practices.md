# UI/UX Best Practices - Boti Dashboard

The Boti management interface must be "modern, minimalist, and very intuitive for non-technical users."

## 1. Visual Excellence
- **Minimalist Aesthetic**: Use ample whitespace, high-quality typography (e.g., Inter or Roboto), and a cohesive color palette.
- **Glassmorphism**: Subtle use of transparency and blurs for a premium feel.
- **Micro-animations**: Smooth transitions for loading states, message updates, and line status changes.

## 2. Intuitive Navigation
- **Dashboard Overview**: At-a-glance status of all connected lines and active AI usage.
- **Simplified Config**: Use toggles and clear labels for AI selection and context parameters.
- **Visual Feedback**: Clear indicators for "Connected", "Reconnecting", and "Error" states in WhatsApp lines.

## 3. Accessibility
- **Non-Technical Focus**: Avoid jargon in the UI. Use "AI Personality" instead of "System Prompt".
- **Responsive Design**: Ensure full functionality across mobile and desktop browsers.
